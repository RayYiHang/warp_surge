## 📋 安装步骤

### 1. 安装 Surge 模块
```
模块链接: https://raw.githubusercontent.com/RayYiHang/warp_surge/refs/heads/main/warp-account-manager-com.sgmodule
```

**安装路径:**
- Surge → 模块管理 → 添加模块 → 粘贴链接 → 确定

### 2. 配置 MitM
```
设置 → HTTPS解密 → 开启MitM → 配置主机名
主机名: app.warp.dev, dataplane.rudderstack.com, securetoken.googleapis.com
```

### 3. 信任证书
```
设置 → HTTPS解密 → 证书 → 安装证书
```

## 🚨 常见问题排查

### 问题1: 模块显示"无效"
**解决方案:**
```
1. 检查 Surge 版本 (需要 >= 4.0)
2. 确认网络连接正常
3. 尝试使用简化版本模块
4. 使用本地文件安装方式
```

### 问题2: 无法访问 warpmanager.com
**解决方案:**
```
1. 检查 Host 配置是否生效
2. 确认 URL Rewrite 规则正确
3. 验证 HTTP Request 脚本是否加载
4. 查看 Surge 控制台错误信息
```

### 问题3: 页面显示空白
**解决方案:**
```
1. 检查脚本路径是否正确
2. 确认 GitHub Raw 链接可访问
3. 验证响应头设置
4. 查看浏览器控制台错误
```

### 问题4: 功能无法使用
**解决方案:**
```
1. 确认 MitM 配置正确
2. 检查持久化存储权限
3. 验证 API 接口响应
4. 重新加载模块
```
